## 帖子喜欢列表

```
GET /group-posts/:post/likes

```

### 参数

| 名称 | 类型 | 说明 |
|:----:|:-----|:-----|
|limit|integer| 默认 15 ，数据返回条数 默认为15|
|after|integer|默认 0 ，翻页标识。|

### 响应

```
status 200
```
